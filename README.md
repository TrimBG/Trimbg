# TrimBG
A fully custom, API-free tool for removing image backgrounds using pure JavaScript.

## Features

- **100% Offline** - No internet or API required
- **Smart Auto-Detection** - Automatically detects and removes backgrounds
- **Optimized Algorithm** - Advanced color distance calculation
- **Easy Download** - Save processed images instantly
- **Drag & Drop** - Drag images directly into the app
- **Fast Processing** - No limits, no waiting
- **Desktop App** - Electron-based desktop application for Linux and Windows

## Clone the repository

```bash
git clone https://github.com/Rayiumir/Trimbg.git
cd Trimbg/
npm install
npm start
```
